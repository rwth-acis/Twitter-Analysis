Project info

A Cloud-Based Near Real-Time Event Analysis Framework for Twitter is an open source framework for real-time analysis of twitter data.

The project is a Master Thesis project from the RWTH Aachen University Informatik 5 chair in contribution with e-dynamics GmbH.

Author:Alexandra Chueva, master student of Software Systems Engineering program
Supervisors:Prof. Dr. Matthias Jarke, RWTH Aachen University
PD Dr. Ralf Klamma, RWTH Aachen University
Advisors:Zinayida Kensche, M.Sc., RWTH Aachen University
